# DesktopFlyouts documentation

DesktopFlyouts provides XAML controls for showing desktop-hosted flyouts from tray icons or from application code.

## Start here

- [Getting started](getting-started.md): install the package and open a flyout.
- [DesktopFlyout](desktop-flyout.md): panel-style flyouts made from one or more islands.
- [DesktopMenuFlyout](desktop-menu-flyout.md): context menus hosted at screen coordinates.
- [SystemTrayIcon](system-tray-icon.md): optional tray icon helper.
- [Focus and activation](focus-and-activation.md): choosing activation behavior.

## Package targets

- `0x5BFA.DesktopFlyouts.WinUI`: Windows App SDK / WinUI 3.
- `0x5BFA.DesktopFlyouts.Uwp`: UWP / WinUI 2.

Both packages expose the main APIs under the `U5BFA.Libraries` namespace.
